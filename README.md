# minimal-react-jest
A minimal project on react and jest

## Features

* React 16
* Webpack 4
* Babel
* Jest
* Staging ES Next Features
* Hot Module Replacement

## Installation

* `git clone git@github.com:jineeshjohn/minimal-react-jest.git`
* cd minimal-react-jest
* npm install
* npm start
* visit `http://localhost:8080/`
